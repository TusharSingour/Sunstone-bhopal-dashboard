E-Learning Student Analytics Dashboard

Yeh project students ki academic performance ko track karne ke liye design kiya gaya hai. Isme ek hi screen par attendance, grades, aur course progress ka pura data visual form (Charts) mein dikhaya jata hai.

Project Highlights

Real-time Progress: Live tracking of completed modules and pending assignments.
Data Visualization: Using Charts/Graphs to show marks and attendance trends.
Mobile Responsive: Students can check their status on any device smoothly.
Profile Management: Easy access to certificates and academic history.

Technical Stack:

React.js Chart.js REST API CSS Grid
