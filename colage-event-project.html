College Event Portal - Management System

Yeh ek digital platform hai jo campus events ko organize aur manage karne ke liye banaya gaya hai. Isse students asani se aane wale events ki jankari le sakte hain aur register kar sakte hain.

Key Features (Khas Baatein):

  • **Event Registration:** Online form bharke seat reserve karein.
  • **Live Updates:** Event ki date aur time ka turant pata chale.
  • **Digital Passes:** Registration ke baad QR-code ya confirmation milna.
  • **Responsive Design:** Mobile aur Laptop dono par smoothly chalta hai.

Project Technologies:

HTML5 CSS3 JavaScript
