College Event Portal

React JS Firebase Tailwind CSS

About the Project

Ye portal college events ko manage karne ka ek single platform hai. Students isme upcoming fests, workshops aur seminars ke liye register kar sakte hain aur organizers registrations ko monitor kar sakte hain.

Key Features

  • Online Registration & E-Tickets
  • Event Schedule & Speaker Profiles
  • Real-time Notification Updates
  • Admin Panel for Attendance Tracking
